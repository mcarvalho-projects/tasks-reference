React (Vite) (CHEAT REFERENCE, simple but working)

Setup:
- Create:
  npm create vite@latest react-tasks -- --template react
  cd react-tasks
  npm install
- Replace:
  src/App.jsx  (use App.jsx from this folder)
- Run:
  npm run dev

Backend URL used:
- http://localhost:5101

CORS note:
- If your backend blocks the browser, enable CORS for localhost in the backend.
